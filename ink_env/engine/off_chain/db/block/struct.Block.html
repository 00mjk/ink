pub fn random<T>(&self, subject: &[u8]) -> Result<T::Hash, OffChainError> where
    T: Environment, [src]

Returns a randomized hash.

Note

  • This is the off-chain environment implementation of crate::random. It provides the same behaviour in that it will likely yield the same hash for the same subjects within the same block (or execution context).

  • Returned hashes on the surface might appear random, however for testability purposes the actual implementation is quite simple and computes those "random" hashes by wrapping XOR of the internal entry hash with the eventually repeated sequence of the subject buffer.
