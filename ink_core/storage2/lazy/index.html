ink_core::storage2

[−][src]Module ink_core::storage2::lazy

Low-level collections and data structures to manage storage entities in the persisted contract storage.

Users should generally avoid using these collections directly in their contracts and should instead adhere to the high-level collections found in crate::storage2::collections. The low-level collections are mainly used as building blocks for internals of other higher-level storage collections.

These low-level collections are not aware of the elements they manage thus extra care has to be taken when operating directly on them.
