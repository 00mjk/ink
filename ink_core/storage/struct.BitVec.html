pub fn iter(&self) -> Iter[src]

Returns an iterator over all bits of the bit vector.

Note

Uncontrained iteration in smart contracts is most often bad practice and should generally be avoided. Consider using this iteration only in the presence of a constrained context, like bitvec.iter().take(5) or similar.

pub fn first_set_position(&self) -> Option<u32>[src]

Returns the position of the first set bit in self if any.

Note

This is a lot more efficient than to naively iterate through all bits of the bit vector.

Complexity

The worst-case time complexity of this procedure is linear with respect to the length of self.
