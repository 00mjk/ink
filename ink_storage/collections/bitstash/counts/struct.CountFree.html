pub struct CountFree {
    counts: [u8; 32],
    full: FullMask,
}

Stores the number of set bits for each 256-bits block in a compact u8.

Fields

counts: [u8; 32]

Set bits per 256-bit chunk.

full: FullMask

Since with u8 can only count up to 255 but there might be the need to count up to 256 bits for 256-bit chunks we need to store one extra bit per counter to determine filled chunks.
